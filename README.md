# Portfolio — Olivier Hollebeke

Portfolio personnel développé avec Vue.js et Vite.  
Il présente mon profil, mes compétences, mes projets et mes informations de contact.

## Stack
- Vue.js (Vite)
- JavaScript
- HTML / CSS
- Bootstrap

## Fonctionnalités
- Site one-page avec navigation par ancres
- Sections : À propos, Formations, Compétences, Infos, Projets, Contact
- Téléchargement du CV en PDF
- Design responsive
